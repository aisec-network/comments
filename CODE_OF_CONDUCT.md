# Contributor Covenant Code of Conduct

## Our Standards

This comment section is for discussion of AI security topics covered by the aisec-network portfolio sites.

**Welcome:**
- On-topic questions and discussions about AI security, ML safety, LLM vulnerabilities
- Constructive feedback on articles
- Sharing relevant resources and research

**Not welcome:**
- Spam, self-promotion, or unsolicited advertising
- Off-topic content
- Harassment or abuse
- Misinformation

## Enforcement

Violations will result in comment removal. Repeat violations will result in blocking from the organization.

Report issues via the GitHub **Report content** button or by opening an issue in this repository.
