# aisec-network/comments

Shared [Giscus](https://giscus.app) comment backend for all aisec-network portfolio sites.

## Overview

All comments across the 26+ AI-security portfolio sites (adversarialml.dev, aisec.blog, jailbreakdb.com, etc.) are stored here as GitHub Discussions under the **General** category.

Each page maps to a discussion by URL. To find comments for a specific page, search Discussions by the page URL.

## Moderation policy

Comments are moderated to keep discussions on-topic and free of spam.

**Automatic controls (active)**
- Interaction limit: **existing users only** — GitHub accounts < 30 days old cannot post (expires 2026-11-12, renewable)
- GitHub native spam detection applies to all Discussion activity

**Manual moderation**
- Spam, off-topic promotions, and abusive comments will be minimized or deleted
- Repeat offenders will be blocked from the organization
- Legitimate but off-topic comments may be moved to an appropriate category

## Spam patterns to watch for

- Casino/gambling/pharma link spam (common in AI-topic comment sections)
- SEO link drops ("great post! visit my site at X")
- Bot-generated compliments with unrelated links
- Crypto/NFT promotions
- Off-topic recruitment/job posts

## Reporting spam

If you see a spam comment, use GitHub's built-in **Report content** button on the discussion, or open an issue in this repo.
